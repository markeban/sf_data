# SfData


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'sf_data'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sf_data

## Usage

To Get all San Francisco film locations:

    $ SfData::FilmLocation.all
